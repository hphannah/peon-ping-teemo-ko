## Installation

### Option 1: From the OpenPeon Registry (recommended)

Make sure [peon-ping](https://github.com/PeonPing/peon-ping) is installed first, then run:

```bash
peon packs install teemo-kr
peon packs use teemo-kr
```

### Option 2: Local Install

1. Clone this repository:

```bash
git clone https://github.com/hphannah/peon-ping-teemo-ko.git
cd peon-ping-teemo-ko
```

2. Install the pack locally:

```bash
peon packs install-local .
peon packs use teemo-kr
```

---

## Switching Packs

```bash
peon packs use teemo-kr     # switch to this pack
peon packs next             # cycle to the next pack
peon packs list             # list all installed packs
```
